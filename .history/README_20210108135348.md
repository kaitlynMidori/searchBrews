
# searchBrews API

Search for National Parks by state

https://kaitlynmidori.github.io/searchBrews/

## Getting Started

1. Input 1 or more state abbreviations
2. Enter desired number of results up to 10
3. Click 'Find Parks!'

* JavaScript
